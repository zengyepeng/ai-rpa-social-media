# 系统架构设计

## 整体架构

采用**AI Agent + RPA 双引擎混合架构**，将决策层与执行层分离：

- **决策层**（云服务器）：大语言模型驱动，负责内容规划、文案生成、效果分析、策略调整
- **执行层**（本地电脑）：影刀RPA驱动，负责账号登录、内容发布、互动回复、数据回传

这种架构的优势：
1. **安全**：影刀操作真实UI，平台无法检测为机器行为
2. **可扩展**：决策层可同时调度多个RPA客户端
3. **维护方便**：UI变化只需更新影刀流程，AI层无需修改

## 数据流

```
时间线 → AI Agent分析数据 → 生成内容策略
     ↓
  内容存入任务队列
     ↓
  Agent通过HTTP通知影刀
     ↓
  影刀执行发布/互动
     ↓
  结果回传给Agent
     ↓
  Agent分析效果 → 调整策略
```

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| AI模型 | DeepSeek API | 内容生成、数据分析 |
| Agent框架 | OpenClaw | 任务编排、流程调度 |
| RPA工具 | 影刀 | 桌面UI自动化执行 |
| 通信 | HTTP API / Webhook | 服务器↔影刀双向通信 |
| 数据库 | SQLite | 数据存储 |
| 部署 | Docker | 服务器端容器化 |
