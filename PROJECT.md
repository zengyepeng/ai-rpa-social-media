# AI Agent + 影刀RPA 社交媒体智能运营系统

## 项目概述

融合AI大模型与影刀RPA的混合架构，实现小红书等社交媒体的智能内容运营。
AI Agent（OpenClaw服务器）负责内容生成、数据分析、决策调度；
影刀RPA（本地电脑）负责执行UI操作，模拟真人行为完成发布、互动。

## 技术架构

```
┌─────────────────────┐      HTTP/Webhook      ┌──────────────────┐
│   AI Agent 大脑      │ ◄────────────────────► │   影刀 RPA 手     │
│  （香港服务器）       │                         │  （本地电脑）      │
│                     │                         │                  │
│  OpenClaw平台        │   指令：发布文案/回复    │  登录小红书       │
│  DeepSeek API       │   状态：成功/失败/异常    │  填写内容         │
│  内容生成引擎         │                         │  上传图片         │
│  数据分析模块         │                         │  点击发布         │
│  调度决策模块         │                         │  回复评论         │
│  SQLite数据库         │                         │  数据回传         │
└─────────────────────┘                         └──────────────────┘
```

## 模块划分

### 1. AI内容生成模块（服务器端）
- 基于DeepSeek API + 提示词工程
- 小红书爆款文案生成（标题、正文、标签、emoji）
- 内容风格模板库
- 批量内容规划

### 2. 数据分析与决策模块（服务器端）
- 账号数据追踪（阅读、点赞、评论、涨粉）
- 爆款内容特征分析
- 发布策略决策（发什么、什么时候发）

### 3. RPA执行模块（影刀 - 本地）
- 小红书自动登录
- 内容发布流程（图文/视频）
- 评论自动回复
- 数据回传服务器

### 4. 通信调度模块
- OpenClaw Agent流程编排
- HTTP API + Webhook 双向通信
- 任务队列与重试机制
- 异常处理（登录失效、验证码）
